## QuantumMail

### Request an Organization

Submit your org request. QuantumMail SuperAdmin will approve and send an Admin setup link.

* Company name
* Organization name
* Your name
* Your email
* Notes (optional)

Submit request

### Join an existing org

You need an Org ID + invite code from your Admin.

* Org ID
* Invite Code
* Username
* Password
* Email (recommended)

Create account (Join)

You’ll be routed based on your role.

* Org ID
* Username
* Password
